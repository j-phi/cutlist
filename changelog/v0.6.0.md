# v0.6.0 — 2026-04-27

_Since [v0.5.1] (2026-04-27)._

## Features

- **Homepage redesign** — replaced the SVG board-layout hero with a static product preview image and reworked the demo flow into a single "Try the demo" button that fetches `demo.cutlist` on click. The `useDemoSeed` composable and its on-startup seeding path are gone.
- **BOM table polish** — rename/edit/delete actions are now inline icon buttons in the name cell (the dedicated action column is gone), dimension column headers show the active unit (`T (mm)`, `W (in)`, etc.), and the grain-lock control is a chip that shows the locked direction with a one-click clear when locked.

## Reliability

- **Hard reload after database reset** — `Reset database` now reloads to `/` after clearing IndexedDB instead of mutating in-memory state in place, so the app re-initialises from a clean slate.

## Breaking Changes

- **License change** — Cutlist is now licensed under the **GNU GPL v3**, replacing the previous MIT license.
