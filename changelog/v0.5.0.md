# v0.5.0 — 2026-04-27

_Since [v0.4.0] (2026-04-27)._

## Features

- **Units on new project** — `NewProjectDialog` now lets you pick millimetres or inches at creation time, replacing the implicit global default.
- **Batch part rename** — rename grouped parts in one go from the BOM colour mapping panel.
- **`.cutlist` file extension** — exports/imports now use `.cutlist` instead of `.cutlist.gz` (still gzip-compressed under the hood).
- **Imports land at the end of the project list** — newly imported projects append to the list rather than jumping to the top.
- **Mobile BOM split panel** — the BOM/preview split is now resizable on phones, with persisted size and orientation-aware sizing.

## Bug Fixes

- **Double-click rename focus** — double-clicking a part name in the BOM now reliably focuses the rename input.
- **BOM PDF printout** — minor column/spacing fixes in the BOM PDF export.

## Code Quality

- **Vue refactor** — extracted `RichTextEditor` into a `useTiptapEditor` composable; split BOM into `BomEmptyState`, `BomModelsList`, and `useBomImport`; moved the project history menu into its own `ProjectHistoryMenu`. `BomTab` and `ProjectTopBar` are roughly a third of their previous size.
- **Layout view simplification** — removed the unplaced-parts visualisation and `LayoutLeftoverList`; the layout tab now focuses on placed boards only.
- **Compact model mode removed** — dropped the `ModelTab` compact toggle.
- **Test coverage expansion** — ~4,700 lines of new component tests across BOM, layout, project tabs, workspace, top bar, and dialogs, plus unit tests for `useBomImport`, `useTiptapEditor`, `usePersistedSplitPanel`, and `formatArchivedDate`.
